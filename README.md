Market Basket Analysis is a data mining technique used to find associations between items that customers buy together.
Features
Find frequent itemsets
Generate association rules
Calculate:
Support
Confidence
Lift
Efficient implementation using optimized algorithms

📊 Example
Input Transactions:
T1: {milk, bread}
T2: {milk, diaper, beer}
T3: {milk, bread, diaper}
T4: {bread, butter}
Output:
Frequent Itemsets:
{milk, bread}
{milk, diaper}
Association Rules:
milk → bread
diaper → milk
